Get Started

Navbar WITHOUT Flexbox (using float + old-school methods)

Important things in this version:
• float: left / float: right → places logo left, menu & button right
• clear: both (via ::after) → prevents parent collapse (same problem as float tutorial)
• Mobile menu uses position: fixed + transform to slide in
• Hamburger still uses checkbox hack (pure CSS toggle)
• Vertical centering with line-height instead of align-items

How to test it

  • Big screen → logo left, links + button right (using float)
  • Resize smaller than ~768px → hamburger appears, menu slides from right
  • Click hamburger → menu opens / closes (no JavaScript!)
