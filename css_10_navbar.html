CSS Flexbox Navbar – Modern & Responsive (2026 Style)

Key Flexbox features used here:
• display: flex on .navbar → horizontal layout
• justify-content: space-between → logo left, menu right
• align-items: center → vertical centering
• gap: 32px → clean spacing between links (no margins!)
• flex-direction: column on mobile menu
• Media query + checkbox hack → pure CSS hamburger (no JS)

How this is better than float-based navbars

  • No collapsing parent problems
  • Easy vertical + horizontal centering
  • gap property = cleaner code
  • Order & flex-grow/shrink available if needed
  • Responsive without weird hacks

Try it yourself

Resize your browser window (or open on phone) to see:

  • Desktop → horizontal menu + button
  • Tablet (~900px) → still horizontal, smaller gaps
  • Mobile (<768px) → hamburger appears, menu slides in
